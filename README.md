# PackScript Beta
### Purpose
- To create a simple website with concise grammar.
- To easily develop games on websites as well.
- To create programs in various fields on the website.
- The purpose is to enable even novice web developers to easily create simple programs.

### How to use
- Enter this code into your terminal. (It currently only works in the terminal)
```
node pack.js yourfile.pk
```

### Caution
- 1. You need Node.js.
- 2. Only for Windows OS.
- 3. It can only be run in the terminal. (A website version will be created soon.)
